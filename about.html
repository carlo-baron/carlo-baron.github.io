About the Project

I, Richard Carlo S. Baron, created the project for the 2nd-semester Midterms performance task while in 11-Nazareth (S.T.E.M.). Additionally, I programmed the website and hosted it using GitHub.

The content of this blog promotes tourist spots in Mindanao, and it includes hyperlinks for 1) viewing more images and information, 2) checking travel packages, and 3) viewing available flights to Mindanao.
